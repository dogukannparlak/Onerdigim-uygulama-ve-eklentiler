# GoFullPage

### 📱 GoFullPage - Full Page Screen Capture

**9 milyon kullanıcının tercihi - En güvenilir tam sayfa ekran görüntüsü uzantısı**

Mevcut sayfanızın tamamının ekran görüntüsünü güvenilir şekilde yakalar ve hiçbir ek izin gerektirmez. Karmaşık sayfalar, scrollable elementler ve embedded iframe’leri destekleyen gelişmiş teknoloji.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Avantaj |
| --- | --- | --- |
| **Tam Sayfa Yakalama** | Sayfanın tamamını scrollable şekilde yakalar | Ek izin gerektirmez |
| **Gelişmiş Teknoloji** | İç scrollable elementler ve iframe desteği | Diğer uzantılardan üstün |
| **Çoklu Format** | PNG, JPEG, PDF export | Esnek kullanım |
| **Premium Editor** | Crop, annotate, emoji ekleme | Profesyonel düzenleme |
| **Otomatik Assembly** | Parçaları otomatik birleştirir | Sorunsuz sonuç |

### 🔧 Çalışma Teknolojisi

**Gelişmiş sayfa yakalama:**

```
📜 Sayfanın her görünür bölümüne scroll yapar
🔧 Parçaları hızlıca birleştirir
🖼️ Inner scrollable elementleri destekler
📱 Embedded iframe'leri yakalar
⚡ Chrome'un sınırlarını aşar
🎯 Büyük sayfaları otomatik böler
```

### 🎨 Premium Editor Özellikleri

**Yeni premium editör ile:**

```
✂️ Crop - İstediğiniz alanı kesin
✏️ Annotate - Metin ve şekil ekleyin
😊 Emoji - Favori emojilerinizi ekleyin
🎨 Tema seçenekleri - Karanlık/aydınlık
📅 Timestamp - Otomatik tarih ekleme
🔄 Auto-scroll - Crop sırasında otomatik kaydırma
```

### 📤 Export Seçenekleri

| Format | Kullanım Alanı | Özellik |
| --- | --- | --- |
| **PNG** | Web ve sosyal medya | Kayıpsız kalite |
