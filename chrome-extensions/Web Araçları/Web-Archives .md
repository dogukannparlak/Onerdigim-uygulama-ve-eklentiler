# Web-Archives

### 🗄️ Web Archives

**50,000+ kullanıcının tercihi - Kapsamlı web arşiv erişim uzantısı**

Web sayfalarının arşivlenmiş ve önbelleğe alınmış versiyonlarını Wayback Machine, Archive.is ve diğer arama motorlarında bulmanızı sağlar. Context menü ve tarayıcı araç çubuğundan arama başlatabilirsiniz.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Kullanım |
| --- | --- | --- |
| **Çoklu Arşiv Desteği** | Wayback Machine, Archive.is ve daha fazlası | Kapsamlı arama |
| **Context Menu** | Sağ tık ile hızlı arşiv arama | Pratik erişim |
| **Toolbar Search** | Tarayıcı araç çubuğundan arama | Direkt erişim |
| **Açık Kaynak** | GitHub’da tamamen şeffaf kod | Güvenilir |
| **Özelleştirilebilir** | Arşiv kaynaklarını aktif/pasif yapma | Kişisel tercih |

### 🌐 Desteklenen Arşiv Servisleri

**Çeşitli arşiv ve cache kaynakları:**

```
🏛️ Wayback Machine - Internet Archive'ın resmi servisi
📚 Archive.is - Hızlı anlık arşivleme
🔍 Google Cache - Google'ın önbellek versiyonu
📖 Bing Cache - Microsoft'un cache servisi
🌐 Yandex Cache - Yandex arama motoru cache'i
📄 Gigablast Cache - Alternatif cache servisi
🔄 Memento Protocol - Akademik arşiv protokolü
Ve daha fazlası...
```

### 🔧 Arama Modları

**Farklı kullanım senaryoları için:**

```
📑 Tab Mode - Aktif sekmedeki sayfayı ara (varsayılan)
🔗 URL Mode - Özel URL adresi ara
🖱️ Context Menu - Link üzerinde sağ tık ile ara
⚡ Toolbar - Araç çubuğu butonuyla ara
```
