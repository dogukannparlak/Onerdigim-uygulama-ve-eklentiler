# Violentmonkey

### 🔧 Violentmonkey

**900,000+ kullanıcının tercihi - Açık kaynak userscript yöneticisi**

WebExtensions API’sini kullanan gelişmiş kullanıcı betiği yöneticisi. Greasemonkey ve Tampermonkey betiklerinin çoğunu destekler ve web sitelerinin çalışma şeklini özelleştirmenizi sağlar.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Avantaj |
| --- | --- | --- |
| **Universal Browser Support** | WebExtensions destekli tüm tarayıcılar | Chrome, Firefox, Edge uyumluluğu |
| **Greasemonkey/Tampermonkey Uyumu** | Mevcut betiklerin çoğunu destekler | Kolay geçiş |
| **Otomatik Güncelleme** | Meta veriye göre otomatik günceller | Her zaman güncel betikler |
| **Gelişmiş Betik Yönetimi** | Sıralama, aktif/pasif, import/export | Organize kontrol |
| **GM Fonksiyonları** | Tam Greasemonkey API desteği | Gelişmiş betik özellikler |

### 🚀 Kurulum ve Başlangıç

**Kolay 5 dakikalık kurulum:**

```
1. Chrome Web Store'dan "Violentmonkey" indir
2. "Chrome'a Ekle" butonuna tıkla
3. Uzantı simgesine tıkla → "Dashboard" aç
4. "+" butonuyla yeni betik ekle veya Greasy Fork'tan indir
5. Betikleri aktifleştir ve web'de keyif al!
```
