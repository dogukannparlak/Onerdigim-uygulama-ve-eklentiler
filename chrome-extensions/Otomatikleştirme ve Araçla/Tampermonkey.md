# Tampermonkey

### Tampermonkey

**11 milyon+ kullanıcının tercihi - Web’i istediğiniz gibi değiştirin**

Tampermonkey, kullanıcı betikleri (userscripts) ile web sitelerini özelleştirmenizi sağlayan en popüler browser uzantısıdır. 72,000+ değerlendirme ile 4.7/5 puan alan bu güçlü araç, web deneyiminizi tamamen kişiselleştirmenize olanak tanır.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Performans |
| --- | --- | --- |
| **Userscript Yönetimi** | Betikleri kolayca yönetin ve düzenleyin | Anlık işlem |
| **2-Tıkla Aktifleştirme** | Betikleri hızlıca açın/kapatın | Ultra hızlı |
| **Senkronizasyon** | Chrome Sync + bulut depoları | Otomatik |
| **Yedekleme/Geri Yükleme** | ZIP dosyaları ve bulut desteği | Güvenli |
| **Otomatik Güncelleme** | Betikler kendiliğinden güncellenir | Sorunsuz deneyim |
| **Çoklu Platform** | 28 dil desteği | Global erişim |

### 🚀 Kurulum ve Temel Kullanım

**5 adımda başlangıç:**

```
1. Chrome Web Store'dan "Tampermonkey" indir
2. www.userscript.zone'u ziyaret et
3. İhtiyacınıza uygun script'i seç
4. "Install" butonuna tıkla
5. Script otomatik aktif olur - web deneyiminiz değişir
```
