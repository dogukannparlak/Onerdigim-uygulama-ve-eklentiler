# WhatFont

### 🔤 WhatFont - Font Tanımlama Aracı

**2 milyon kullanıcının tercihi - Web sitelerinde font keşfetmenin en kolay yolu**

Chengyinliu.com tarafından geliştirilen WhatFont, web sitelerinde kullanılan fontları tanımlamanın en basit yoludur. Geliştirici araçlarının karmaşıklığına elveda deyin - sadece metin üzerine fareyi getirin ve fontları keşfedin.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Kullanım |
| --- | --- | --- |
| **Hover Font Detection** | Fare hover ile anında font tanıma | Geliştirici araçları gerektirmez |
| **Click for Details** | Tıklayarak detaylı stil bilgisi | Font-family, size, weight, color |
| **Font Service Detection** | Typekit, Google Fonts, FontDeck tanıma | Web font servis analizi |
| **Color Detection** | Metin rengi analizi | Hex, RGB değerleri |
| **Escape Key Exit** | ESC tuşuyla hızlı çıkış | Kolay kullanım |
| **Modern UI** | Güncellenmiş arayüz tasarımı | Site stillerinden etkilenmez |

### 🛠️ Çalışma Teknolojisi

**3 adımda font keşfi:**

```
1. 🎯 Uzantı simgesine tıklayın - "WhatFont?" modu aktif
2. 🖱️ İncelemek istediğiniz metne fareyi getirin - font adı görünür
3. 👆 Daha fazla detay için metne tıklayın - tüm CSS bilgileri
4. ⌨️ ESC tuşu ile çıkış yapın veya simgeye tekrar tıklayın
```

### 🎨 Desteklenen Font Bilgileri

**Detaylı CSS style bilgileri:**

```
🔤 Font-family - Arial, Helvetica, Georgia vb.
📏 Font-size - Piksel, em, rem değerleri
⚖️ Font-weight - Normal, bold, 100-900 arası değerler
📐 Line-height - Satır aralığı bilgisi
🎨 Color - Hex, RGB, RGBA renk kodları
💡 CSS properties - Tüm text ile ilgili özellikler
```

### 🚀 Kurulum ve Kullanım

**3 adımda font uzmanı olun:**

```
1. Chrome Web Store'dan "WhatFont" indir
2. İncelemek istediğiniz web sitesine git
3. Uzantı simgesine tıkla ve font keşfetmeye başla
4. Fareyi metinlere getir - anında font bilgisi
5. Detaylar için tıkla - tam CSS style analizi
```
