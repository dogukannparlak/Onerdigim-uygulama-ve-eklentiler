# Immersive-Translate

### 🌍 Immersive Translate - Bilingual Website & PDF Translation

**10 milyon kullanıcının tercihi - AI destekli iki dilli web sayfası ve PDF çeviri uzantısı**

Web sitelerini, PDF’leri, EPUB e-kitapları ve video altyazılarını gerçek zamanlı olarak iki dilli çeviren ücretsiz tarayıcı uzantısı. DeepL, OpenAI ChatGPT, Google Translate dahil 10+ çeviri motorunu destekler.
