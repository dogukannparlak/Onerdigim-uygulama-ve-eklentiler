# Medium-Parser

### 📖 Medium Parser

**9,000+ kullanıcının tercihi - Medium makalelerini ücretsiz okuyun**

Medium tabanlı makaleleri ücretsiz okuma imkanı sunan tarayıcı uzantısı. Member-only makaleleri Google Web Cache veya Archive.is aracılığıyla erişilebilir hale getirir ve harici yayıncılardan makaleleri bulup tam içeriğe ulaşmanızı sağlar.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Teknoloji |
| --- | --- | --- |
| **Paywall Bypass** | Member-only makaleleri ücretsiz okuma | Google Web Cache |
| **Archive.is Desteği** | Alternatif arşiv erişimi | Web arşivleme |
| **Harici Yayıncı Bulma** | Google search ile tam içerik bulma | Search API |
| **Açık Kaynak** | Şeffaf ve denetlenebilir kod | GitHub mevcut |
| **Hızlı Erişim** | Tek tıkla makale erişimi | Optimize edilmiş |

### 🛠️ Çalışma Mekanizması

**3 farklı yöntemle makale erişimi:**

```
1. 🌐 Google Web Cache - Önbelleğe alınmış sürümler
2. 📚 Archive.is - Web arşiv hizmeti
3. 🔍 Google Search - Harici yayıncı bulma
4. ⚡ Otomatik fallback - Çalışmayan metot varsa geçiş
```
