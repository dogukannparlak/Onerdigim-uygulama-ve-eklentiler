# YouTube-Playlist-Duration-Calculator

### 🕐 YouTube Playlist Duration Calculator

**40,000+ kullanıcının tercihi - Playlist analiz ve sıralama uzantısı**

YouTube oynatma listelerinin toplam süresini hesaplar ve küçük playlistleri farklı kriterlere göre sıralar. Eğitim içeriklerini planlama ve zaman yönetimi için mükemmel araç.

### 🎯 Ana Özellikler

| Özellik | Açıklama | Limit |
| --- | --- | --- |
| **Toplam Süre Hesaplama** | Playlist’in tam uzunluğunu gösterir | Sınırsız video |
| **Akıllı Sıralama** | Küçük playlistleri farklı kriterlere göre sıralar | 100 video |
| **Süre Bazlı Sıralama** | Video sürelerine göre artan/azalan | Organize izleme |
| **Görüntülenme Sıralaması** | İzlenme sayısına göre sıralama | Popülerlik analizi |
| **Yükleme Tarihi Sıralaması** | En yeni/eski videolar | Zaman takibi |
| **Kanal Adı Sıralaması** | Alfabetik creator sıralaması | Content creator analizi |

### 🔧 Kullanım Yöntemi

**3 adımda playlist analizi:**

```
1. YouTube playlist sayfasına git (youtube.com/playlist?list=...)
2. Sol taraftaki playlist bilgi panelinde toplam süreyi gör
3. Küçük playlistlerde sıralama seçeneklerini kullan
```

### 📊 Görüntülenen Bilgiler

**Detaylı playlist analizi:**

```
⏰ Total Duration - Toplam süre (hh:mm:ss formatında)
✅ Videos Counted - Hesaplamada kullanılan video sayısı
❌ Videos Not Counted - Hesaplanamayan video sayısı
📈 Ortalama video süresi (otomatik hesaplama)
📋 Playlist video sayısı
🎯 İzleme süresi tahminleri
```

### 🔧 Sıralama Özellikleri (≤100 Video)

**Küçük playlistler için gelişmiş sıralama:**

```
⏰ Duration - Video sürelerine göre
   📈 Ascending: Kısa → Uzun
   📉 Descending: Uzun → Kısa

📅 Upload Date - Yükleme tarihine göre
   🆕 Newest First: En yeni videolar
   📜 Oldest First: En eski videolar

🎬 Channel Name - Kanal adına göre
   🔤 A → Z: Alfabetik sıralama
   🔤 Z → A: Ters alfabetik
```

### 🚀 Kurulum ve Kullanım

**Hızlı başlangıç:**

```
1. Chrome Web Store'dan "YouTube Playlist Duration Calculator" indir
2. YouTube'da herhangi bir playlist sayfasına git
3. Sol panelde toplam süre bilgisini gör
4. 100 video altı playlistlerde sıralama seçeneklerini kullan
5. Farklı playlist'lerde analiz yaparak zaman planlaması yap
```

### ⚠️ Sınırlamalar ve Notlar

```
📊 Sıralama sadece 100 video altı playlistlerde çalışır
❌ Private ve deleted videolar hesaplamaya dahil edilmez
⚡ Çok büyük playlistlerde yükleme süresi artabilir
🔄 YouTube API değişikliklerinde güncelleme gerekebilir
📱 Mobile YouTube app'te çalışmaz (sadece web)
🎯 Playlist URL'si 'youtube.com/playlist' ile başlamalı
```
